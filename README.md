# 📊 Data Analytics Internship Task – KPI & Cohort Analysis

## 🔷 Project Overview

This project focuses on solving a business problem by performing **data analysis** and deriving meaningful insights using **Key Performance Indicators (KPIs)** and **Cohort Analysis**.

The objective is to evaluate business performance, customer behavior, and retention patterns.

---

## 🔷 Key Features

* ✅ KPI Calculation
* ✅ Customer Behavior Analysis
* ✅ Cohort Analysis (Retention Study)
* ✅ Automated Python Script
* ✅ Error-handling for missing/empty data

---

## 🔷 KPIs Defined

### 1. Conversion Rate

**Formula:**
Conversion Rate = Completed Orders / Total Orders

**Purpose:**
Measures the efficiency of order completion.

---

### 2. Average Order Value (AOV)

**Formula:**
Average Order Value = Total Revenue / Number of Orders

**Purpose:**
Indicates average spending per order.

---

### 3. Customer Churn Rate

**Formula:**
Churn Rate = 1 - (Retained Customers / Total Customers)

**Purpose:**
Measures customer retention and loyalty.

---

## 🔷 Deep-Dive Analysis: Cohort Analysis

Cohort Analysis groups customers based on their **first purchase month** and tracks their activity over time.

### 🔍 Insights:

* Helps identify **customer retention trends**
* Shows how different customer groups behave over time
* Helps businesses improve **customer engagement strategies**
